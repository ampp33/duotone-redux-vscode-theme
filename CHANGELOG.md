## v1.0.0

- Added Sky Theme
- Added Earth Theme
- Added Forest Theme
- Added Space Theme
- Themed the integrated terminal's ANSI colors (previously undefined, falling back to VS Code defaults) using a strict two-hue formula derived from each variant's own warm/cool palette
- Themed git decorations, editor diagnostics (error/warning/info), diff editor highlighting, and the bracket-highlight "unexpected bracket" color
- Themed floating widgets: hover, suggest, and peek view popups
- Themed notifications, inputs, list/quick-pick selection and highlight colors, links, and the activity bar badge
- Added visible borders between adjacent same-background panels (sidebar, status bar, editor group, title bar, terminal panel), derived per-theme from each variant's own palette
- Moved the active editor tab's accent border from top to bottom
- Restyled the primary button (e.g. Source Control "Commit") for better contrast against the theme background
